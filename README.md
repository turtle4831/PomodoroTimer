# Pomodoro Study Timer

Local desktop Pomodoro timer built with **PyQt6**.

## Project layout

```
client/     PyQt6 desktop app
tests/      Unit tests
```

## Setup

```bash
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```

## Run the app

From the project root:

```bash
python -m client.main
```

## Run tests

```bash
pytest
```
